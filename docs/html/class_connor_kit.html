Detailed Description

ConnorKit class, containing all of the coolest functions! Over the years I've had to roll many of my own Arduino functions to solve some interesting or tedious problems, and I've decided to put all of these solutions into a single, easy-to-use Arduino library. Don't worry - the only flash and RAM space it takes are the functions you actually use! The compiler is smart enough to ignore everything else.

  • Connor Nishijima

§ ConnorKit()

ConnorKit::ConnorKit ( )

Used to initialize the ConnorKit library. Example usage before setup() would be: 

ConnorKit ck;

"ck" can be anything you want, as long as it's reflected in the rest of your code. Because each of these functions are wrapped in the ConnorKit class, you'll use them like this: 

ck.array_sum(arr,len);
